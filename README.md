## :gb: English

### Overview

Interactive web application to explore and test 30+ Prompt Engineering frameworks. Features 3D pushable cards, multi-provider AI support, and custom signature frameworks (CALIBRO, BCM, PRISMA).

### Features

- **30+ Frameworks** - APE, AIDA, BAB, CARE, Chain of Thought, Tree of Thoughts
- **3D Pushable UI** - Interactive cards with shadow, edge, and front layers
- **Multi-Provider** - OpenAI, Google Gemini, ZAI support
- **Dark/Light Mode** - Theme switching with localStorage persistence
- **16K Token Responses** - Extended AI response context
- **i18n Support** - Italian and English languages

### Framework Categories

| Category | Examples |
|----------|----------|
| Fundamentals | APE, BAB, PAR, RTF |
| Business | AIDA, SMART, STAR |
| Creative | CRAFT, ROSES, SPARC |
| Advanced | Chain-of-Thought, Tree-of-Thoughts |
| Signature | CALIBRO (GPT-5), BCM (GLM-4.6), PRISMA (Claude) |

### Quick Start

```bash
git clone https://github.com/fracabu/promptify-vue.git
cd promptify-vue
npm install
npm run dev
```

Access at `http://localhost:5174`

Applicazione web interattiva per esplorare e testare 30+ framework di Prompt Engineering. Card 3D pushable, supporto multi-provider AI e framework signature personalizzati (CALIBRO, BCM, PRISMA).
